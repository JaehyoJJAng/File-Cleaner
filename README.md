# File-Cleaner


<h1>How to install</h1>
<pre>
<code>
$ git clone https://github.com/JaehyoJJAng/File-Cleaner.git
$ cd File-Cleaner
</code>
</pre>

<h1>Usage</h1>
<pre>
<code>
$ python3 file-cleaner.py
</code>
</pre>


<p>
  <strong>
    Replace the path of self.target_dir in file-cleaner.py with your desired path.
  </strong>
</p>
